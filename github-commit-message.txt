تحسين واجهة تسجيل الدخول وإضافة الشعار الجديد

تم إجراء التعديلات التالية:
- إضافة الشعار البرتقالي الجديد بتصميم محسن
- تحسين شكل وحجم الشعار في صفحة تسجيل الدخول
- إضافة تأثيرات خلفية منحنية للصفحة
- تحسين شكل حقول الإدخال بزوايا دائرية
- إضافة تأثيرات تفاعلية لزر تسجيل الدخول
- تحسين تنسيق النصوص والتباعد بين العناصر

تساعد هذه التغييرات على تحسين تجربة المستخدم وجعل الشعار الجديد يتناسب بشكل أفضل مع تصميم التطبيق.
